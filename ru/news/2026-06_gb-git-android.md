# Май

Первоначально в мае я хотел сделать отображение истории трат в Git Budget, но
история за собой потянула необходимость хранения данных. А данные ведь надо
хранить в Git! Поэтому я сделал поддержку Pull/Push жёстко зашитого репозитория
из git.opengamestudio.org в Git Budget. И это замечательно работает. Пользуюсь
на своём телефоне (вот видео вверху).

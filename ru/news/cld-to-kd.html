Рождение Диалекта Котлина

2026-03-07

Февраль

В феврале планировал запустить GitBudget на Desktop с помощью Qt, но кроличья нора Kotlin Multiplatform + C++ оказалась глубже первоначальных оценок. Мне удалось проверить техническую осуществимость задуманной связки, но результат будет лишь в марте.

В ходе путешествия по кроличьей норе я осознал необходимость переквалифицировать проект Cross-language dialect в Kotlin dialect, чтобы сосредоточиться на Kotlin Multiplatform. В связи с этим в феврале также создал KD на основе CLD, чтобы упростить его запуск путём создания приложения Node.js вместо JVM, а также генерацию полей F.

Март

В марте продолжу доработку Kotlin Dialect для работы с C++, чтобы наконец уже запустить GitBudget на Desktop с помощью Qt.

Сайт сгенерирован ПСКОВОМ из этого исходного кода.

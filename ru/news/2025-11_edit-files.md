# Редактирование файлов проекта

В октябре ПСКОВ 2 получил следующие новые возможности:

| № | Функциональность | Описание | Фото |
| --- | --- | --- | --- |
| 1 | Вкладка `Files` | Содержит список файлов проекта (было без вкладки) | |
| 2 | Вкладка `Edit` | Содержит редактор Ace для изменения выбранного во вкладке `Files` файла (новое) | |
| 3 | Вкладка `Render` | Будет содержать результат после конвертации файла Markdown (будет) | |

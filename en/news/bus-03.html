The first Bus pattern implementation

I spent January and February making Bus pattern implementation for iOS. The end result looks nice to me.

I have also done load testing. Turns out, the Bus is 92 times more expensive than a function call. Nonetheless, this doesn't mean Bus is bad. On the contrary, the numbers highlight the Bus should be used in architectural glue code where it can save up to 30% of code
